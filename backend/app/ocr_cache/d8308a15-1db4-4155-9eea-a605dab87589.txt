CNN (Convolutional Neural Network) 
What is CNN (Convolutional Neural Network)? 
CNN ek Deep Learning algorithm hai jo specially images, videos aur visual data ko analyze 
karne ke liye use hota hai. 
Ye image ke andar se important features (edges, shapes, patterns) automatically learn karta 
hai aur prediction karta hai. 
Example: 
 
Input: Dog image  
 
Output: “Dog (99%) 
 
Advantages of CNN 
1.  Automatic Feature Extraction 
CNN khud features (edges, shapes, textures) learn karta hai — manual feature engineering nahi 
karni padti. 
2. Fewer Parameters (compared to ANN) 
Weight sharing aur local connectivity ki wajah se parameters kam hote hain → efficient model. 
3. High Accuracy in Image Tasks 
Image classification, detection me CNN bahut high accuracy deta hai. 
4.  Spatial Information Preserve karta hai 
Image ka structure (2D relation) maintain rehta hai. 
5.  Better Performance on Large Data 
Large image datasets par CNN best perform karta hai. 
 
Disadvantages of CNN 
1. High Computation Cost 
Training ke liye GPU ki zarurat hoti hai. 
2. Slow Training 
Deep CNN models ko train karne me time lagta hai. 
3.  Large Dataset Requirement 
Achha performance ke liye huge labeled dataset chahiye. 
 
4.  Complex Architecture 
CNN ko design aur tune karna difficult hota hai (filters, layers, etc). 
 
 Limitations of CNN 
1.  Not good for non-image data 
Tabular data (Excel type) ke liye CNN useful nahi hai. 
2.  Needs huge computing power 
GPU/TPU ke bina training slow ho jati hai. 
3.  Overfitting risk 
Small dataset par CNN easily overfit ho jata hai. 
4.  Black box model 
Explain karna difficult hota hai ki model ne decision kaise liya.

CNN Architecture (Full Flow) 
CNN ka architecture ek step-by-step pipeline hota hai jisme image ko gradually simple se 
complex features me convert kiya jata hai. 
 
 Overall CNN Architecture 
Input Image 
     ↓ 
Convolution Layer 
     ↓ 
Activation (ReLU) 
     ↓ 
Pooling Layer 
     ↓ 
Convolution Layer (repeat) 
     ↓ 
Activation (ReLU) 
     ↓ 
Pooling Layer 
     ↓ 
Flatten Layer 
     ↓ 
Fully Connected (Dense Layer) 
     ↓ 
Output Layer (Softmax / Sigmoid) 
 
 Step-by-Step Explanation 
 
1. Input Layer 
 Raw image CNN ko input diya jata hai. 
Example: 
28 × 28 × 1 (grayscale) 
224 × 224 × 3 (RGB) 
 Image pixel values me convert hoti hai. 
 
 2. Convolution Layer (Core of CNN) 
 Yahi CNN ka brain hota hai. 
Kaam: 
 
Image se features extract karna  
 
Jaise edges, corners, shapes  
How? 
Filter / Kernel use hota hai: 
Image + Filter → Feature Map 
 Example: 
 
Vertical edge detector  
 
Horizontal edge detector  
 
 3. Activation Function (ReLU) 
Non-linearity add karta hai 
ReLU formula: 
f(x) = max(0, x) 
Why? 
 
Negative values remove  
 
Model ko complex patterns seekhne deta hai  
 
  
4. Pooling Layer 
 Image size reduce karta hai (downsampling) 
Types: 
 
Max Pooling (most used)  
 
Average Pooling  
Example: 
2x2 block → max value pick 
 Benefits: 
 
Computation reduce  
 
Overfitting control  
 
Important features retain  
 
5. Repeated Conv + Pooling 
CNN me ek hi layer nahi hoti. 
Structure: 
Conv → ReLU → Pool 
Conv → ReLU → Pool 
 Early layers: 
 
edges detect  
 Deep layers: 
 
face, object detect  
 
6. Flatten Layer 
 2D feature map ko 1D vector me convert karta hai 
Example: 
Feature Map: 
7 × 7 × 64 
 
↓ 
 
Flatten: 
 
3136 × 1 vector 
 
 7. Fully Connected Layer (Dense Layer) 
 Ye actual decision maker hota hai 
 
All neurons connected  
 
Final reasoning karta hai  
Example: 
Features → Dense Layer → Class prediction 
 
 8. Output Layer 
 Final prediction deta hai 
Types: 
 Softmax (Multi-class) 
Example: 
 
Cat   
 
Dog   
 
Horse   
Cat: 70% 
Dog: 20% 
Horse: 10% 
 
 Sigmoid (Binary) 
Example: 
 
Cancer / No Cancer  
0 → No 
1 → Yes 
 
 
# Import libraries 
from tensorflow.keras.models import Sequential 
from tensorflow.keras.layers import Conv2D, MaxPooling2D, Flatten, Dense 
 
# CNN Model Design 
model = Sequential() 
 
# Step 1: Convolutional Layer 
model.add(Conv2D( 
    filters=32,              # number of filters 
    kernel_size=(3,3),       # filter size 
    activation='relu',       # activation function 
    input_shape=(28,28,1)    # input shape (MNIST grayscale images) 
)) 
 
# Step 2: Pooling Layer 
model.add(MaxPooling2D(pool_size=(2,2))) 
 
# Step 3: Flatten Layer 
model.add(Flatten()) 
 
# Step 4: Dense Layers 
model.add(Dense(128, activation='relu'))   # hidden dense layer 
model.add(Dense(64, activation='relu'))    # another hidden dense layer 
model.add(Dense(10, activation='softmax')) # output layer (10 classes) 
 
# Step 5: Compile Model 
model.compile( 
    optimizer='adam', 
    loss='categorical_crossentropy', 
    metrics=['accuracy'] 
) 
 
# Summary of the model 
model.summary()

Step 5 : Build CNN Model 
Definition 
Build Model ka matlab CNN ki architecture banana, jisme decide karte hain ki kitni layers hongi 
aur har layer ka kaam kya hoga. 
 
Code 
import tensorflow as tf 
 
# CNN Model 
model = tf.keras.Sequential([ 
 
    # Input Layer 
    tf.keras.layers.Input(shape=(224,224,3)),          # 224x224 RGB Image 
 
    # Convolution Block - 1 
    tf.keras.layers.Conv2D(32, (3,3), activation="relu"),   # 32 Filters 
    tf.keras.layers.MaxPooling2D((2,2)),                    # Down Sampling 
 
    # Convolution Block - 2 
    tf.keras.layers.Conv2D(64, (3,3), activation="relu"),   # 64 Filters 
    tf.keras.layers.MaxPooling2D((2,2)), 
 
    # Convolution Block - 3 
    tf.keras.layers.Conv2D(128, (3,3), activation="relu"),  # 128 Filters 
    tf.keras.layers.MaxPooling2D((2,2)), 
 
    # Convert Feature Map → 1D 
    tf.keras.layers.Flatten(), 
 
    # Fully Connected Layer 
    tf.keras.layers.Dense(128, activation="relu"), 
 
    # Output Layer (Binary Classification) 
    tf.keras.layers.Dense(1, activation="sigmoid") 
 
])
